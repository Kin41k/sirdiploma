# Как запустить CinemaRec — инструкция для новичков

Читай внимательно, делай по шагам, не пропускай ничего.

---

## Шаг 0 — Установи нужные программы

Тебе нужны три программы. Проверь, есть ли они уже:

**Открой командную строку** (нажми `Win + R`, напиши `cmd`, Enter) и введи по очереди:

```
python --version
```
Должно написать что-то вроде `Python 3.11.x`. Если пишет ошибку — скачай Python:
👉 https://www.python.org/downloads/ (кнопка "Download Python 3.x.x")
> ⚠️ При установке обязательно поставь галочку **"Add Python to PATH"**

```
node --version
```
Должно написать `v18.x.x` или выше. Если ошибка — скачай Node.js:
👉 https://nodejs.org (кнопка "LTS")

```
git --version
```
Если ошибка — скачай Git:
👉 https://git-scm.com/downloads

После установки **перезапусти командную строку** и проверь снова.

---

## Шаг 1 — Скачай проект

В командной строке перейди туда, куда хочешь сохранить проект. Например, на рабочий стол:

```
cd %USERPROFILE%\Desktop
```

Затем скачай проект:
```
git clone https://github.com/ССЫЛКА_НА_РЕПОЗИТОРИЙ.git
```

Перейди в папку проекта:
```
cd cinerarec
```

> Вместо `cinerarec` напиши реальное название папки, которая создалась.

---

## Шаг 2 — Создай файлы настроек

Скопируй файлы-примеры (они уже есть в репозитории):

```
copy backend\.env.example backend\.env
copy frontend\.env.local.example frontend\.env.local
```

> Всё, больше ничего в этих файлах менять не нужно — дефолтные настройки подходят.

---

## Шаг 3 — Настрой бэкенд (Python-часть)

**Открой новое окно командной строки** и не закрывай его до конца.

Перейди в папку backend:
```
cd %USERPROFILE%\Desktop\cinerarec\backend
```

Создай виртуальное окружение (изолированная папка для библиотек, чтобы не засорять систему):
```
python -m venv venv
```

Активируй его:
```
venv\Scripts\activate
```
> После этого в начале строки появится `(venv)` — это нормально, так и должно быть.

Установи все необходимые библиотеки:
```
pip install -r requirements.txt
```
> ⏳ Это займёт 3–7 минут, будет много текста — жди пока не вернётся строка ввода.

Создай базу данных:
```
alembic upgrade head
```
> Должно написать несколько строк и закончить без ошибок.

Заполни базу данных фильмами и тестовыми пользователями:
```
python -m app.utils.seed
```
> ⏳ Это самый долгий шаг — 5–10 минут. Будет показывать прогресс типа `...50000 movies imported`. Жди надписи `[DONE] Database seeded successfully.`

---

## Шаг 4 — Обучи модели машинного обучения

**Не закрывай предыдущее окно!** Перейди в папку ml:

```
cd ..
cd ml
```

Запусти обучение:
```
python main.py
```

> ⏳ Займёт 5–15 минут в зависимости от компьютера. Будет показывать прогресс.
> В конце напишет что-то вроде:
> ```
> === Evaluation Results ===
>   precision_at_10: 0.1842
>   recall_at_10: 0.1234
>   ndcg_at_10: 0.2156
>   rmse: 0.8734
> Training complete. Models saved to ml/models/
> ```
> Это означает что всё сработало.

---

## Шаг 5 — Установи зависимости фронтенда

**Открой второе окно командной строки** (не закрывай первое!).

Перейди в папку frontend:
```
cd %USERPROFILE%\Desktop\cinerarec\frontend
```

Установи пакеты:
```
npm install
```
> ⏳ 2–3 минуты, создастся папка `node_modules` — это нормально.

---

## Шаг 6 — Запусти приложение

Теперь нужно запустить два сервера одновременно в двух разных окнах командной строки.

**Окно 1 — Бэкенд** (вернись в первое окно):
```
cd %USERPROFILE%\Desktop\cinerarec\backend
venv\Scripts\activate
uvicorn app.main:app --reload --port 8000
```
> Должно написать `Application startup complete.` и зависнуть — это нормально, сервер работает.

**Окно 2 — Фронтенд** (второе окно):
```
cd %USERPROFILE%\Desktop\cinerarec\frontend
npm run dev
```
> Должно написать `Ready - started server on http://localhost:3000` и зависнуть — это нормально.

---

## Шаг 7 — Открой приложение

Открой браузер и перейди по адресу:

```
http://localhost:3000
```

Готово! Приложение работает 🎉

## Как останавливать серверы

В каждом из двух окон нажми `Ctrl + C` — сервер остановится.

## Как запускать в следующий раз

Шаги 0–5 делаются только один раз. В следующий раз нужно только Шаг 6:

**Окно 1:**
```
cd %USERPROFILE%\Desktop\cinerarec\backend
venv\Scripts\activate
uvicorn app.main:app --reload --port 8000
```

**Окно 2:**
```
cd %USERPROFILE%\Desktop\cinerarec\frontend
npm run dev
```

## Что-то пошло не так?

| Ошибка | Решение |
|--------|---------|
| `python is not recognized` | Переустанови Python с галочкой "Add to PATH" |
| `pip install` падает на `scikit-surprise` | Установи Visual C++ Build Tools: https://visualstudio.microsoft.com/visual-cpp-build-tools/ |
| `alembic upgrade head` пишет ошибку | Убедись что `(venv)` активирован и ты в папке `backend` |
| Сайт не открывается | Убедись что оба сервера запущены (оба окна должны "зависнуть") |
| Порт 8000 занят | Напиши `uvicorn app.main:app --reload --port 8001` и в `frontend\.env.local` измени на `NEXT_PUBLIC_API_URL=http://localhost:8001` |
